注意!

  • 所有命名必须是英文字母，不能带有符号
  • HTML中js引入标签模板必须是src="路径/文件名"，注意；是双引号
  • HTML中css引入标签模板必须是href="路径/文件名"，注意；是双引号
  • HTML中img引入标签模板必须是src="路径/文件名"，注意；是双引号
  • CSS中img,font引入标签模板必须是url('路径/文件名')，注意；是单引号
  • 静态资源文件结构js、css、img、font、media文件夹是同级单独目录并且不能有子目录
  • js文件路径需要自己在JS文件中手动替换
  • media文件路径需要自己在HTML页面中手动替换
